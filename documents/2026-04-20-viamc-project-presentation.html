UC2 · Engagement Model

Two Delivery Options

Offer A · Build it (Managed Project)

MC team builds the Python automation + redaction script as a delivered solution for VIAMC.

  • Faster time to result
  • Full ownership by MC during build
  • Ongoing maintenance dependency

Offer B · Workshop (On-Premise)

MC runs an on-prem workshop with VIAMC staff — co-discover the solution together, teach them to handle it.

  • Lower MC effort & VIAMC cost
  • High knowledge transfer
  • VIAMC self-sufficient afterwards

Use Case 3

AI Tool Centralization

Staff are spread across 12 GenAI tools with no central control. Goal: centralize governance & audit what data leaves the org.

Gemini / Bard GROK ChatGPT Perplexity NotebookLM Canva ElevenLabs Adobe Express CapCut Hailuo Wrtn Deepl
Reframe: this is a CASB (Cloud Access Security Broker) problem — a well-understood enterprise security pattern.
UC3 · Solution Options

Three Approaches

A · CASB

Netskope · Zscaler · MS Defender

Pros

  • Shadow-IT discovery
  • DLP + redaction built-in
  • Audit log out of box

Cons

  • High licence cost
Recommended

B · Proxy + DLP

Squid + custom DLP rules

Pros

  • Cheaper than CASB
  • Flexible rule-set

Cons

  • More setup & maintenance
  • No shadow-IT discovery
Budget-friendly

C · Browser Agent

Managed browser extension

Pros

  • Low infra lift
  • Fast to deploy

Cons

  • Easy to bypass
  • Misses API calls
Light-touch
Recommendation: CASB (A) if budget allows; Proxy + allowlist (B) as pragmatic middle ground.
