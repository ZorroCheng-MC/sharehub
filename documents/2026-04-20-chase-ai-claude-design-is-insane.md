## 📝 Notes & Key Takeaways

### Main Insights

1. **Claude Design is available at `claude.ai/design`** — web only (no desktop app yet). Requires Pro Max or Enterprise plan. Built with Opus 4.7.

2. **Four creation modes**: Prototype, Slide Deck, Template, Other. Each prompts a plan-mode style Q&A before generating — similar to Claude Code's plan mode but visual.

3. **Brand design system** — paste your GitHub URL or drag a local folder; Claude Design pulls brand assets (colors, typography, logos) automatically to ensure design consistency.

4. **Visual editor is genuinely granular** — select individual elements, change colors/sizes directly in the UI. Adds comments, draw annotations, and sends them straight to Claude for iteration. Comparable to Cursor's editor or Lovable.

5. **Export pipeline** — ZIP, PDF, PowerPoint (editable in MS Office), Canva, or hand off to Claude Code with a ready-made command. This bridges design-to-code workflow without leaving the Anthropic ecosystem.

6. **Key positioning insight**: Claude Design isn't just Canva. Think of it as Google AI Studio for front-end design — a visual medium that translates visual intent into natural language, which becomes code, which becomes a rendered visual. The visual-first loop eliminates the "describe in text what I want visually" bottleneck.
