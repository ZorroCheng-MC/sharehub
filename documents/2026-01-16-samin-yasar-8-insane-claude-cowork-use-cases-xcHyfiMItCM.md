### Main Insights

1. **Claude Cowork is Claude Code's agentic technology made accessible** - Same underlying tech but with a simpler interface for non-developers
2. **Multi-tasking capability** - Can run multiple tasks simultaneously via the sidebar
3. **File system access** - Claude can organize, read, and work with local files when given permission
4. **Browser control** - Full browser automation to navigate sites, fill forms, download files
5. **Teachable workflows** - Record your browser actions and Claude learns to repeat them
6. **Skills ecosystem** - Extensible through custom skills and community-contributed "Awesome Claude Skills" on GitHub
7. **Slidev integration** - Pro tip: use Slidev for more resilient presentation generation with better asset handling
