# Chinese Language Exercise - 褒義詞與貶義詞

Source: OCR from PDF

---

## 一、選出適當的貶義詞,把代表字母填在橫線上。

1. 這家網店高價出售置牌貨,歎騙顧客,營商手法(  )
   - A. 低微
   - B. 高明
   - C. 惡劣
   - D. 優異

2. 經過幾年的相處,我們終於知道他只是(  ),不是真誠待人。
   - A. 笑裏藏刀
   - B. 笑面迎人
   - C. 笑容可掬
   - D. 惡名昭彰

3. 有人假裝問路,想在叔叔警覺性降低時偷走錢包,幸好被叔叔識破他的(  )。
   - A. 策略
   - B. 攻略
   - C. 計劃
   - D. 詭計

4. 我初次在大眾面前演講,心情非常緊張,說話時(  )的,雙手不停地顫抖。
   - A. 隨心所欲
   - B. 倒行逆施
   - C. 前仆後繼
   - D. 顛三倒四

## 二、選出適當的褒義詞和貶義詞,填在橫線上。

選項組:
- 精明 / 糊塗
- 無事生非 / 安守本分
- 無所事事 / 日理萬機
- 節儉 / 奢侈
- 遺臭萬年 / 流芳百世
- 高瞻遠矚 / 目光如豆

填空題:

1. 雖然張先生家境富裕,可是他為人既老實又______,從來不會買______的物品。

2. 八十多歲的爺爺沒有因年紀而變得______,每當我和弟弟發生爭執時,他總能______地判斷誰是誰非。

3. 郭卜虞原本沒有甚麼人生目標,終日______!但自從他跟朋友合夥經營一家餐廳後,生活變得充實,每天______。

4. 偉仔自小被家人寵壞了,經常______,他的雙親一方面很自責,另一方面很感慨,不知道兒子何時才願意______。

## 三、填空題

三國時期,劉備知人善任,世人對他加以褒揚,令他______。
他的兒子劉禪繼位後卻庸碌無能,結果______。

---

**註:** 三國時期,魏指佔據北方的曹魏,吳指江東一帶的孫吳,蜀漢指______。
