# ChatGPT Personal Styling Analysis Prompts (Hair / Makeup / Color)

Upload a portrait photo to ChatGPT (GPT-4o Image) and paste the prompts one at a time. Each produces a high-resolution Traditional Chinese styling analysis infographic.

## Prompts

```
1.髮型分析
請根據我上傳的人像照片，製作一張高感個人髮型分析圖卡。保留主角原本五官、型與真實特徵，透過左右或並排對比方式，展示不同髮型套用在主角身上的效果，清楚區分[最適合]、[普通]與[不建議]髮型，讓人一眼看出哪些髮型最修飾臉型、提升氣質與整體顏值。可比較長髮、短髮、瀏海、捲髮、直髮、層次剪裁、綁髮造型等。版面設計需乾淨時尚、像專業造型顧問報告，整體以視覺呈現為主，只使用簡短標籤，不要加入長段文字。高解析度，資訊清楚，適合社群分享。

2.妝容分析
請根據我上傳的人像照片，製作一張高質感[適合的妝容分析]資訊圖表。包括色系、妝容風格等，整體為中文版本，風格乾淨時尚、像美妝雜誌專。以原人物五官為基礎，保留真實長相與辨識度，不過度修圖。版面採視覺優先設計，重點用圖像呈現，文字精簡，不要長段落。

3.色彩分析
請根據我上傳的人像照片，製作一張高質感個人色彩分析圖卡。保留主角原本五官、膚色、臉型與真實特徵，透過左右或並排對比方式，展示不同服裝顏色穿在主角身上的效果，清楚區分[適合色]與[不適合色]，讓人一眼看出哪些顏色最提升膚色、提升氣色與整體質感。版面設計需乾淨時尚、像專業形象顧問報告，整體以視覺呈現為主，只使用簡短標籤(例如:推薦、普通、避免)，不要加入長段文字。高解析度，繁體中文、資訊清楚。
```
