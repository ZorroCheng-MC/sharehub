### Main Insights

1. **Character Consistency Formula**: Nano Banana Pro achieves perfect consistency when given proper reference material + photorealistic directives + detailed physical descriptions

2. **The Secret Prompt Approach**: The "secret prompt" mentioned in the title provides a structured framework for:
   - Maintaining exact facial features across generations
   - Preserving body proportions
   - Keeping consistent lighting and camera angle
   - Ensuring photorealistic output (avoiding 3D render or CGI appearance)

3. **Photorealism Over Stylization**: All examples emphasize documentary/photography style over cinematic or artistic rendering. This is critical for consistency.

4. **Three Key Character Sheet Types**:
   - With reference photo (easiest consistency)
   - From description only (more challenging, requires detailed prompting)
   - Wardrobe variations (same person, different clothes)

5. **Preservation of Natural Imperfection**: The generator should NOT "fix" asymmetries or imperfections - these actually IMPROVE character consistency by making the character look like a real person

### Actionable Points

1. **For Your Next Project**: Use the provided free prompts as a template base. Customize the physical descriptors for your specific character.

2. **Test Your Consistency**: Generate a batch of 4 full-body + 3 portrait shots. Compare facial features pixel-by-pixel to ensure identity preservation.

3. **Lighting Documentation**: Explicitly specify "soft window light" or "neutral studio lighting" in your prompt. Avoid cinematic lighting descriptions.

4. **Wardrobe Changes**: When editing, include "same person, same day, different clothes" to guide the model to preserve identity while changing only garments.

5. **Reference Photo Strategy**: If you have a good reference, use it. If not, write extremely detailed physical descriptions (include asymmetries, natural skin variations, age lines, etc.).

6. **Iterate on Drafts**: The first generation might not be perfect. Use refinement prompts to adjust specific features while preserving overall identity.
