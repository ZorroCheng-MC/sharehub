# 蒜蓉杏仁炒菜心

## 🥬 材料

| 食材 | 份量 |
|------|------|
| 菜心（或芥蘭） | 300g |
| 蒜頭 | 3-4瓣 |
| 杏仁片 | 2湯匙 |
| 食油 | 2湯匙 |
| 鹽 | 適量 |
| 糖 | 少許 |
| 紹興酒（可選） | 1茶匙 |

## 👨‍🍳 做法

### 1. 準備材料
- 菜心洗淨，瀝乾水分
- 蒜頭切薄片
- 杏仁片備用

### 2. 炒蒜片
- 中火熱鑊，落油
- 放入**一半蒜片**，炒至金黃香脆
- 撈起備用（做裝飾用）

### 3. 炒菜心
- 用剩返嘅油，轉**大火**
- 放入另一半蒜片爆香
- 放入菜心，快炒
- 灑少許紹興酒（提香）
- 加鹽、糖調味
- 炒至菜心軟身但保持翠綠（約2-3分鐘）

### 4. 上碟
- 將菜心盛起放碟
- 灑上炸蒜片
- 灑上杏仁片
- 完成！

## 💡 小貼士

✅ **大火快炒** 保持翠綠色  
✅ **蒜片分兩次用** — 一次炸香做裝飾，一次爆香  
✅ **杏仁片** 可以預先烤香更香口  
✅ **唔好炒太耐** — 菜會出水變黃

## 🌱 碳排放分析

呢道菜好環保！

| 食材 | 碳排放 |
|------|--------|
| 蔬菜 | 約 2 kg CO₂/kg |
| 杏仁 | 約 5-8 kg CO₂/kg（用量少） |
| 煮食能源 | 約 0.5-1 kg CO₂ |

**總碳排放：約 0.5-1 kg CO₂**

💚 比起同等份量嘅牛肉低 **10倍以上**！  
（牛肉約 5-10 kg CO₂ 同等份量）

- 蛋白質
- 維他命 E
