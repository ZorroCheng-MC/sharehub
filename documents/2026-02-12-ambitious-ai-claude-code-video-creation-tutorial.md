## 📖 Description

This tutorial demonstrates how to use Claude Code with the Remotion skill to create professional videos, animations, and presentations through simple text prompts. The video covers three levels: basic setup, creating videos with custom design systems, and advanced features like adding voiceovers and stitching multiple clips together.

## 📝 Notes & Key Takeaways

### Main Insights

1. **Claude Code + Remotion = Video Generation**: Claude can now code video content using the Remotion framework, enabling creation of motion graphics, animations, and complete presentations through text prompts.

2. **Design System Integration**: You can extract design systems from existing websites and Claude will automatically apply those styles (colors, fonts, spacing) to generated videos.

3. **Three-Level Approach**: The tutorial follows a progressive learning path - setup, basic video creation, then advanced features like audio and multi-clip editing.

4. **Frame-Based Editing**: Videos can be edited by referencing specific frame numbers, making it easy to update individual components without regenerating entire videos.

5. **AI Voiceover Pipeline**: Key AI + ElevenLabs integration allows automatic generation of synchronized voiceovers that align with visual components.

### Actionable Points

- Download Claude Code from claude.ai and install on Mac or Windows
- Create a dedicated project folder (e.g., "Claude/Claude-video")
- Install NodeJS from nodejs.org before setting up Remotion
- Use the prompt: "add the remotion skill with that command and update the claude markdown document"
- Extract brand design systems with: "review this website and create a design system aligned with the website"
- Generate videos with: "create a 10-20 second video of my business using the exact style and theme"
- Add voiceovers with: "add a male/female voice over to the video and ensure words align with components"
- Combine clips with: "combine video 1, video 2, etc. into one video and render as MP4"
